# MPNA-SVD

Run with debug mod : 
```bash
cmake -DDEBUG=on ..
make
./Main.exe size_n size_m
```
Run without debug mod : 
```bash
cmake -DDEBUG=off ..
make
./Main.exe size_n size_m
```
Run with benchmark mod : 
```bash
cmake -DBENCHMARK=on ..
make
./Main.exe size_n size_m
```
